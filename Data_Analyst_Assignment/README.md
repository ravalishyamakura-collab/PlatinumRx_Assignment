HOTEL MANAGEMENT SYSTEM (MYSQL)
--------------------------------
This project contains SQL solutions for:
1. Last room booked by each user
2. Total booking amount in November 2021
3. Bills >1000 in October 2021
4. Most & least ordered items per month (2021)
5. Customers with second-highest bill value per month

CLINIC MANAGEMENT SYSTEM (MYSQL)
--------------------------------
This project contains SQL solutions for:
1. Revenue by sales channel
2. Top 10 customers
3. Month-wise revenue, expenses, profit & status
4. Most profitable clinic per city
5. Second least profitable clinic per state
